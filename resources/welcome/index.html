Azure IoT Hub Extension

Interact with Azure IoT Hub, IoT Device Management, IoT Edge Management, IoT Hub Device Simulation, IoT Hub Code Generation.

Quick Starts

Select IoT Hub if you have existing one

Click below button to select IoT Hub via command, or expand for alternative instructions.

Select IoT Hub
To select IoT Hub, click above button, or follow below steps.
  1. In Explorer of VS Code, click "Azure IoT Hub" tab in the bottom left corner.
  2. Click "Select IoT Hub" in context menu.
  3. If you have not signed in to Azure, a pop-up will show in the bottom right corner to let you sign in to Azure.
  4. After you sign in, your Azure Subscription list will be shown, then select Azure Subscription and IoT Hub.
  5. The devices and endpoints list will be shown in "Azure IoT Hub" tab in a few seconds.

Create IoT Hub if you don't have one

Click below button to create IoT Hub via command, or expand for alternative instructions.

Create IoT Hub
To create IoT Hub, click above button, or follow below steps.
  1. In Explorer of VS Code, click "Azure IoT Hub" tab in the bottom left corner.
  2. Click "Create IoT Hub" in context menu.
  3. If you have not signed in to Azure, a pop-up will show in the bottom right corner to let you sign in to Azure.
  4. After you sign in, select subscription, resource group, location, pricing tier and enter the name of your IoT Hub.
  5. IoT Hub will be created in a few minutes.

Say Hello to Azure IoT Hub in 1 Minute

  • Register a device
  • Say Hello to IoT Hub (Send device-to-cloud message)
  • Monitor device-to-cloud (D2C) message
  • Generate Azure IoT application

Register a device

A device must be registered with your IoT hub before it can connect.
  1. Click ... > Create Device at AZURE IOT HUB tab, or type Azure IoT Hub: Create Device in Command Palette.
  2. Enter device ID and press Enter.
  3. Wait a few seconds until the new device is created.

Say Hello to IoT Hub (Send device-to-cloud message)

  1. Right-click your device and select Send D2C message to IoT Hub
  2. Enter the message count, interval and message body. Click Send
  3. Results will be shown in OUTPUT > Send D2C Messages view.

Monitor device-to-cloud (D2C) message

While you could send message to your IoT Hub, it is also possible to monitor those message in VS Code.
  1. Right-click your device and select Start Monitoring Built-in Event Endpoint.
  1. The monitored messages will be shown in OUTPUT > Azure IoT Hub view.
  2. To stop monitoring, right-click the OUTPUT view and select Stop Monitoring Built-in Event Endpoint.

Generate Azure IoT application

You could also use your favorite programing language (e.g. C#, Go, Java, Node.js, PHP, Python and Ruby) to quickly generate an Azure IoT application.
  1. Right-click your device and select Generate Code
  2. In language list, select Node.js
  3. In code template list, select Send device-to-cloud message
  4. The JavaScript will be generated immediately. The device connection string is also automatically filled in.
